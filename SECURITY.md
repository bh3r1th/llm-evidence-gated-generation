# Security Policy

## Supported Versions

This repository is currently a PoC (`v0.1-ega-poc` line). Security fixes are best-effort.
Only the latest tagged version is expected to receive updates.

## Reporting a Vulnerability

Please report vulnerabilities privately to:
- `security@REPLACE_ME` (placeholder)

Do not open a public issue for undisclosed vulnerabilities.

If available, use GitHub Security Advisories for coordinated disclosure.

## What to Include

- A clear description of the issue.
- Reproduction steps and affected commands/files.
- Impact assessment (confidentiality/integrity/availability).
- Any known mitigations or temporary workarounds.

## Response Expectations

We will acknowledge reports and triage them, but no fixed response SLA is promised.
