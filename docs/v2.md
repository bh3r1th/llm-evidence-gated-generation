# EGA v2 Workflow

## Recommended POC Configuration

- Verifier model: `MoritzLaurer/DeBERTa-v3-base-mnli-fever-anli`
- Reranker model: `cross-encoder/ms-marco-MiniLM-L-6-v2`
- Accept threshold: `0.05`
- Conformal state: regenerate from exported calibration rows under the current verifier semantics
- Safe-answer rendering: enabled for final examples and docs
- Budget mode: experimental, excluded from headline results

`0.05` is the recommended operating point for the current pilot artifact. If the verifier path or score semantics change, rerun calibration export and conformal fitting before reporting new results.

## Recommended Command Sequence

1. Threshold sweep

```bash
ega threshold-sweep \
  --dataset demos/v2/eval_dataset_pilot.jsonl \
  --out runs/v2_compare/eval/threshold_sweep.json \
  --accept-threshold 0.05
```

2. Export calibration rows

```bash
ega export-calibration-rows \
  --dataset demos/v2/eval_dataset_pilot.jsonl \
  --out runs/v2_compare/calibration/pilot_calibration_rows.jsonl \
  --accept-threshold 0.05
```

3. Fit conformal state

```bash
ega conformal-calibrate \
  --in runs/v2_compare/calibration/pilot_calibration_rows.jsonl \
  --out runs/v2_compare/calibration/pilot_conformal_state.json \
  --epsilon 0.05 \
  --min-calib 1
```

4. Run the final comparison

```bash
ega v2-eval \
  --dataset demos/v2/eval_dataset_pilot.jsonl \
  --out runs/v2_compare/eval/pilot_threshold_005_recalibrated.json \
  --conformal-state runs/v2_compare/calibration/pilot_conformal_state.json \
  --accept-threshold 0.05 \
  --reranker-model cross-encoder/ms-marco-MiniLM-L-6-v2 \
  --render-safe-answer
```

5. Generate publish artifacts

```bash
ega generate-poc-report \
  --source-summary runs/v2_compare/eval/pilot_threshold_005_recalibrated.json \
  --dataset demos/v2/eval_dataset_pilot.jsonl \
  --conformal-state runs/v2_compare/calibration/pilot_conformal_state.json \
  --summary-out runs/v2_compare/eval/final_poc_summary.json \
  --report-out docs/poc_results.md
```

## Headline Variants

- `v1_baseline`: verifier only
- `rerank_only`: verifier plus reranker
- `conformal_only`: verifier plus conformal abstention
- `combined`: verifier plus reranker plus conformal abstention

## Experimental Variants

- `budget_only`

If `budget_only` does not run under a binding budget that actually changes evaluated pairs relative to baseline, treat it as experimental only. Do not include it in headline tables.

## Safe-Answer Rendering

`--render-safe-answer` renders a plain-text answer from accepted claims only.

Example:

```text
The Eiffel Tower is in Paris. [e1]
It opened in 1889. [e3]
```

The compact example is also stored in `demos/v2/safe_answer_example.md`.
